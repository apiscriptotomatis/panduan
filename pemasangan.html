Cara Setting Poin

1

Buka Script Di kode editor

  • Trebedit ( Aplikasi Android )
  • VSCODE ( Pc / Laptop )
2

Cari Kode Berikut

  • var namaMembership = "{{nama_membership}}"; var membershipRates = { "Premium": 5500, "Gold": 6500, "Silver": 10000, "Bronze": 25000, };
3

Setting Sesuai Membership yang sudah kamu buat di bukaolshop

  • Nama Membership Harus sama dengan yang tertera di bukaolshop Huruf besar kecil dan spasi berpengaruh
  • Membership Premiun : 5500, Maka Setiap Pembelian Minimal 5500 akan mendapatkan 1 poin
  • Membership Gold : 6500, Maka Setiap Pembelian Minimal 5500 akan mendapatkan 1 poin
  • Membership Silver : 10000, Maka Setiap Pembelian Minimal 5500 akan mendapatkan 1 poin
  • Membership Bronze : 25000, Maka Setiap Pembelian Minimal 5500 akan mendapatkan 1 poin
  • Silahkan ganti nama nama membership sesuai dengan yang kamu buat di aplikasi Bukaolshopmu
  • Jadi logikanya Nama Membership - Minimal Pembelian

Cara Setting Poin Berdasarkan lvl membership dan Membuat Membership

1

Buka Aplikasi Bukaolshop

  • Klik tab Toko
  • Pilih Membership
2

Klik Tanpa +

  • Untuk Membuat Membership
3

Centang Bagian

  • Jadikan Reseller
  • Tambahkan Poin
